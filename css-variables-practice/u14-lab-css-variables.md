### ✅ Success Criteria

Your project is complete when you've:  

1. added three CSS variables representing colors to the **:root** element rule in your style sheet
2. added your first and last name as an h1 to your page
3. added three other facts about yourself -- as `<h3>` elements -- to your page
4. created and applied three separate CSS classes to your `<h3>` elements
5. used a different CSS variable in each CSS class
