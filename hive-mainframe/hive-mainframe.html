The Hivedriod Mainframe: The Main Router

The Hive Mainframe is a location deep inside the code of all Hivedroids, that only Hivedroids, or anything synthesized with the nanodroids the Hive is made of, are able to go into. The synthesized beings, especially humanity, will be locked out after a given amount of time. All Hivedroids are given access to the Mainframe all the time unless I declare otherwise. Hivedroids are all connected in this fashion, and it will always be that way, as long as Hivedroids exist, and I am in control.
